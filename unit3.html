Covariance & Pearson Correlation

This notebook investigates how dataset transformations (noise, scale, offset) affect covariance and correlation strength using NumPy, Matplotlib, and SciPy.

Access

  • ðŸ“¥ Download/View Notebook (.ipynb)

ðŸ“„ Description

The notebook generates a base dataset and applies random noise, offsets, and scaling to simulate new data. Each variation is analyzed using covariance and Pearson's correlation to measure linear relationships.

Code Snapshot

np.random.seed(42)
for i in range(5):
  base = 20 * np.random.randn(1000) + 100
  noise = np.random.randint(1, 40)
  offset = np.random.randint(-100, 100)
  scale = np.random.uniform(0.5, 1.5)

  transformed = base * scale + noise * np.random.randn(1000) + offset
  covariance = np.cov(base, transformed)[0][1]
  corr, _ = pearsonr(base, transformed)
  print(f"Run {i+1}: Covariance = {covariance:.2f}, Pearson = {corr:.4f}")

Results

The correlation values ranged from weak to strong depending on the noise level and scaling. As noise increased, Pearson correlation weakened, demonstrating the effect of data quality on linear relationships.
