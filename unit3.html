Observations & Reflections

Each notebook was run with varied input parameters including sample size, noise, scaling, and feature interaction. Observations were recorded on:

  • How noise affects correlation strength
  • Impact of adding or removing features in regression
  • Detecting underfitting or overfitting in models
  • Visualizing residuals and polynomial curves

Learning Outcomes

  • Understood data quality impacts on model performance
  • Practiced model interpretation and parameter tuning
