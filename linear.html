Linear Regression with cars.csv

The code loads cars.csv, selects Engine Size as the independent variable and CO2 Emissions as the dependent variable, and fits a linear regression model using scikit-learn. A scatter plot with regression line is displayed.

Code Snapshot

import pandas as pd
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

df = pd.read_csv("cars.csv")
X = df[['Engine Size']]
y = df['CO2 Emissions']

model = LinearRegression()
model.fit(X, y)

print("Intercept:", model.intercept_)
print("Slope:", model.coef_[0])

plt.scatter(X, y)
plt.plot(X, model.predict(X), color='red')
plt.xlabel("Engine Size")
plt.ylabel("CO2 Emissions")
plt.title("Linear Regression: Engine Size vs CO2")
plt.show()

Results

The linear regression model shows a moderate linear relationship between engine volume and COâ‚‚ emissions.

  • The slope is rather small but consistent (~0.01), indicating that as engine volume increases, COâ‚‚ emissions increase proportionally.
  • The RÂ² values ranged between 0.25 and 0.38 across different test splits, which suggests that the model captures some, but not all, of the variation in emissions.
  • As expected, increasing the test size slightly reduced model performance due to fewer training points.
  • Higher Mean Squared Error (MSE) values on larger test sets showed that prediction error grows with more unseen data.

This sensitivity analysis confirmed that the model is data-dependent and performs best when the training set is sufficiently large and well distributed.
